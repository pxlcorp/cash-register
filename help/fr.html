Introduction

La Pxl Caisse Enregistreuse vous permet de vendre des articles simplement et rapidement.

Première étape: Connecter un scanner

  • Obtenez un scanner à main
  • Allez dans le menu Options
  • Allez dans l'onglet Scanner
  • Appuyez sur le bouton Ajouter un scanner
  • Prenez le scanner à main
  • Appuyez sur le clique gauche de votre sourie en regardant la machine
  • Ajoutez un scanner fixe (Optionnel)
    • Obtenez un scanner fixe
    • Prenez le scanner à main
    • Appuyez sur le clique gauche de votre sourie en regardant le scanner fixe

Deuxième étape: Ajouter des articles

  • Trouvez un article à vendre
  • Prenez le scanner à main
  • Appuyez sur le clique gauche de votre sourie en regardant l'article
  • Changez les informations de l'article
    • Allez dans le menu Articles
    • Trouvez votre article dans la list (classe:nom du model)
    • Appuyez sur le bouton de modification (le bouton bleu)
    • Entrez le nom et le prix
    • Appuyez sur Confirmer

Troisième étape: Vendre

  • Scannez vos article
    • Prenez le scanner à main
    • Appuyez sur le clique gauche de votre sourie en regardant un article
    • Ou passez l'article devant le scanner fixe
  • Ajoutez un service (Optionnel)
    • Appuyez sur Ajouter un service
    • Entrez la description du service et le prix
    • Appuyez sur Confirmer
  • Attendez que le client termine la transaction
    • Il doit appuyer sur la touche utiliser en regardant la caisse (autre par que l'écran)
    • Il doit sélectionner le mode de paiement
    • Il doit confirmer
  • Imprimez le reçu pour le client (Optionnel)

Quatrième étape: Gestion de l'argent

  • Allez dans le menu Argent
  • Prendre l'argent Cash
    • Appuyez sur le bouton Prendre l'argent
    • Entrez le montant désiré
  • Distribuer le Crédit
    • Sélectionnez le joueur auquel vous voulez donner de l'argent en appuyant sur le bouton à droit de son nom
    • Sélectionnez le mode de transfert
    • Entrez le montant désiré
  • Déposer de l'argent
    • Appuyez sur le bouton Déposer de l'argent
    • Sélectionnez le mode de dépôt
    • Entrez le montant désiré

Cinquième étape: Relier les caisses enregistreuses

  • Allez dans le menu Options
  • Allez dans l'onglet Server
  • Pour héberger un serveur
    • Appuyez sur le bouton editer à droit du mot de passe
    • Entrez un mot de passe
  • Pour rejoindre un serveur
    • Appuyer sur le bouton Rejoindre un serveur
    • Entrez le nom et le mot de passe du serveur

Options

  • Gestion des accès
    • Gérez les personnes autorisées à utiliser la machine.
    • La liste des personnes autorisées est partagé entre chaque machine reliée au même serveur.
  • Option du serveur
    • Le nom sur serveur sert d'identifiant pour les machines voulant se connecter à celui ci.
    • Lorsqu'il n'y a pas de mot de pass, le serveur ne peut plus être rejoint par une nouvelle machine
    • Pour vous connecter, entrez le nom et le mot de passe du serveur en question
  • Gestion des scanners
    • Enregistrer ou supprimer des scanners
  • Les permissions
    • Que le propriétaire peut modifier les permissions.
    • Permet de bloquer des accès pour les employés
